# README #

### Dependências do Sistema ###

* Node.js 4.5.*
* NPM
* Bower
* MongoDB 3.2.*

```
#!bash

npm install -g bower
```

### Instalando dependências do projeto ###

```
#!bash

npm install
```

```
#!bash

bower install
```

### Run ###

```
#!bash

gulp serve
```

### Build ###
```
#!bash

gulp build
```
